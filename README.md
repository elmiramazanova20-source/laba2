Проект: Развертывание статического HTML-сайта в PaaS
Данный проект представляет собой простой статический веб-сайт, созданный для изучения процессов деплоя (развертывания) в облачных PaaS-решениях.

Цель работы
Изучить механизм работы и жизненный цикл приложений в среде PaaS на примере развертывания статического сайта, написанного на HTML.

Технологический стек
Frontend: HTML5, CSS3 (если используется).

Инструменты развертывания: PaaS-платформа  Netlify.

Контроль версий: Git.

Как запустить проект
Локально
Клонируйте репозиторий:


[git clone https://github.com/ваш-логин/название-проекта.git](https://github.com/elmiramazanova20-source)
Откройте файл index.html в любом браузере.

Развертывание в PaaS
Для развертывания этого сайта в облаке выполните следующие шаги:

Загрузите код в публичный или приватный репозиторий на GitHub.

Подключите ваш аккаунт к выбранной PaaS-платформе.

Выберите репозиторий и запустите процесс Deploy.

Платформа автоматически создаст URL-адрес, по которому ваш сайт будет доступен в сети.

 Этапы реализации
 Создание структуры HTML-документа.

Настройка системы контроля версий Git.

 Конфигурация проекта для работы с облачной платформой.

 Успешный деплой и проверка доступности сайта.

 Выводы
В ходе выполнения работы были освоены навыки автоматизации развертывания. Использование PaaS позволяет абстрагироваться от настройки серверов и сосредоточиться на разработке фронтенд-части приложения.
